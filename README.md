# Cosmetic King

This project is a React-based e-commerce application for cosmetic products.

## Features
- **Product Catalog**: Browsable list of cosmetic items.
- **Cart & Checkout**: Functional shopping cart.
- **Admin Panel**: For managing products and stock.
- **Cloudinary Integration**: Upload product images directly to the cloud.
- **Firebase Firestore**: Real-time cloud database.

## Run Locally

1. Install dependencies:
   `npm install`
2. Create `.env` file with Cloudinary credentials.
3. Run the app:
   `npm run dev`
